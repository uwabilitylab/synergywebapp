Space Limitation

Due to this website being a showcase of muscle synergy calculation, server space is limited to 5 unique file uploads per user. This is done in order to maximize the number of users that will have exposure to the website while keeping the cost of the server minimal. If you would like to upload additional files there are resources detailing how to host your own synergy calculator on our github. Feel free to contact Claire Mitchell if you have additional questions.
